# 居家服務訪視紀錄產生器

協助居家服務員快速整理訪視紀錄的網頁工具。填寫表單，即時生成可一鍵複製貼上系統的訪視紀錄文字。

線上版：<https://sun901019.github.io/home-visit-record/>

## 主要功能

- **體況評估**：意識、感官、皮膚、睡眠等多面向勾選
- **健康狀況**：慢性病、營養、管路、飲食質地
- **自理能力**：移位、如廁、沐浴、購物、用藥等
- **環境與家庭脈絡**：輔具、居家安全、家屬支持系統
- **服務計畫**：可建立多筆固定排程＋按需服務
- **服務代碼**：內建 BA/GA/SC 系列代碼，含目標說明
- **即時預覽**：右側即時生成訪視紀錄文字，支援單段／全部複製
- **自動儲存**：表單會自動保存在瀏覽器，意外關掉也不怕
- **歷史紀錄**：可儲存多筆，方便日後回顧或重複使用
- **列印 / PDF**：支援列印模式，輸出乾淨版面

## 本機開發

```bash
npm install
npm run dev
```

開發伺服器網址：<http://localhost:5173/home-visit-record/>

## 建置

```bash
npm run build
npm run preview
```

## 部署

推送到 `main` 分支後，GitHub Actions 會自動建置並部署到 GitHub Pages。

## 技術

- Vite + React 19
- Tailwind CSS v4
- lucide-react 圖示

## 隱私

所有資料只儲存在你的瀏覽器（localStorage），不會上傳任何伺服器。
